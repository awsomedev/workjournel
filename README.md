## macOS Claude Code mode

- macOS chat can use local Claude Code directly through the `claude` CLI.
- The app does not manage Claude tokens.
- Users must install Claude Code CLI and run `claude auth login` on their Mac.
- App rely on the system-installed `claude` command being available in PATH.
